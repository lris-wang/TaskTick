# TaskTick 隐私政策

**最后更新：2026 年 4 月 18 日**

TaskTick（"我们"、"我们的"或"应用"）尊重您的隐私。本隐私政策说明了我们在您使用 TaskTick 应用（涵盖 Web 桌面端、移动端）时如何收集、使用、存储和披露您的信息。

请您仔细阅读本隐私政策。如果您不同意本政策的条款，请不要使用我们的应用。

---

## 一、我们收集的信息

### 1.1 您直接提供的信息

- **账户信息**：注册时提供的邮箱、用户名和密码（密码经过哈希处理，我们无法查看原始密码）
- **个人资料**：头像 URL（可选）
- **任务数据**：您创建的任务、笔记、项目分组、标签及其内容（标题、描述、截止日期、优先级、重复规则、附件等）
- **团队信息**：如果您创建或加入团队，相关的团队名称和成员信息
- **AI 功能**：如果您使用自然语言创建任务的 AI 功能，输入的原始文本

### 1.2 自动收集的信息

- **使用日志**：您与应用交互的基本行为（创建任务、完成时间、错误日志），用于改进产品体验
- **设备信息**：操作系统类型和版本、应用版本、IP地址（用于安全审计）
- **本地数据**：桌面客户端在本地 SQLite 数据库中存储的任务缓存（设备本地加密存储）

> 我们**不**收集您在第三方服务（如 Google Calendar、Notion）中的原始内容。

### 1.3 第三方服务收集的信息

我们使用以下第三方服务，它们根据自身隐私政策收集数据：

| 第三方服务 | 用途 | 隐私政策 |
|-----------|------|---------|
| Supabase / PostgreSQL | 数据库托管 | supabase.com/privacy |
| Cloudflare | CDN 和 HTTPS 加速 | cloudflare.com/privacy |
| GitHub（仅桌面端自动更新） | 推送更新包 | docs.github.com/en/site-policy/privacy-policies |
| Apple / Google（仅应用分发） | 应用分发平台 | apple.com/legal/privacy / policies.google.com/privacy |
| OpenAI API（可选 AI 功能） | 自然语言任务解析 | openai.com/privacy |

## 二、我们如何使用信息

我们使用收集的信息用于：

1. **提供服务**：运行、维护和改进 TaskTick 的核心功能
2. **数据同步**：在您的多设备间同步任务和设置（端到端加密传输）
3. **安全防护**：检测异常登录、防止滥用和垃圾信息
4. **通知提醒**：在任务截止前向您发送提醒（可随时在设置中关闭）
5. **统计分析**：统计用户量、功能使用率等匿名聚合数据，用于产品决策
6. **客服支持**：响应您的问题和技术支持请求

## 三、数据存储和安全

### 3.1 存储地点

- **服务器**：数据存储于 **Supabase（美国 AWS us-east-1 区域）** 的 PostgreSQL 数据库中
- **桌面客户端**：离线数据缓存于您设备的本地数据库，传输前经过 HTTPS 加密

### 3.2 保留期限

- 您的账户数据和任务数据，**直到您主动删除账户**
- 删除账户后，数据在 30 天内从所有备份中清除

### 3.3 安全措施

- 所有网络传输强制使用 **TLS 1.2+** 加密（HTTPS）
- 数据库访问受强密码和基于角色的访问控制（RBAC）限制
- 我们承诺：对任何安全事件，将根据适用法律在 **72 小时内** 通知受影响的用户

## 五、您的权利

您对您的个人数据享有以下权利：

### 5.1 访问和导出

您可以随时在应用内**导出您的全部数据**（支持 JSON 格式），包含任务、笔记、标签和项目信息。

如果您在上述权利之外还需要其他数据操作，请通过以下方式联系我们：

📧 **privacy@tasktick.com**（请在邮件标题注明"隐私请求"）

我们将在 **30 天内** 处理您的请求。

## 七、儿童隐私

TaskTick **不面向 13 岁以下（或所在国法定年龄）以下的儿童**，我们不会故意收集儿童的个人信息。如果发现我们意外收集了儿童信息，我们将立即删除。
